write()

write( arg0,  arg1 = null,  arg2 = null) : \io\vertx\jphp\core\file\AsyncFile

$data [Buffer]

write($data)

Write a io.vertx.core.buffer.Buffer to the file at position {@code position} in the file, asynchronously.

If {@code position} lies outside of the current size of the file, the file will be enlarged to encompass it.

When multiple writes are invoked on the same file there are no guarantees as to order in which those writes actually occur

The handler will be called when the write is complete, or if an error occurs. $buffer [Buffer] the buffer to write $position [integer] the position in the file to write it at $handler [callable] the handler to call when the write is complete write($buffer,$position,$handler)

read()

read( arg0,  arg1,  arg2,  arg3,  arg4) : \io\vertx\jphp\core\file\AsyncFile

Reads {@code length} bytes of data from the file at position {@code position} in the file, asynchronously.

The read data will be written into the specified {@code Buffer buffer} at position {@code offset}.

If data is read past the end of the file then zero bytes will be read.

When multiple reads are invoked on the same file there are no guarantees as to order in which those reads actually occur.

The handler will be called when the close is complete, or if an error occurs. $buffer [Buffer] the buffer to read into $offset [integer] the offset into the buffer where the data will be read $position [integer] the position in the file where to start reading $length [integer] the number of bytes to read $handler [callable] the handler to call when the write is complete read($buffer,$offset,$position,$length,$handler)
