reply()

reply( arg0,  arg1 = null,  arg2 = null) :

Reply to this message.

If the message was sent specifying a reply handler, that handler will be called when it has received a reply. If the message wasn't sent specifying a receipt handler this method does nothing. $message [Object] the message to reply with. reply($message) The same as {@code reply(R message)} but you can specify handler for the reply - i.e. to receive the reply to the reply. $message [Object] the message to reply with. $replyHandler [callable] the reply handler for the reply. reply($message,$replyHandler) Link #reply(Object) but allows you to specify delivery options for the reply. $message [Object] the reply message $options [DeliveryOptions | mixed] the delivery options reply($message,$options) The same as {@code reply(R message, DeliveryOptions)} but you can specify handler for the reply - i.e. to receive the reply to the reply. $message [Object] the reply message $options [DeliveryOptions | mixed] the delivery options $replyHandler [callable] the reply handler for the reply. reply($message,$options,$replyHandler)

fail( arg0,  arg1) :
