binaryValue()

binaryValue() : \io\vertx\jphp\core\buffer\Buffer

Return the binary value.

JSON itself has no notion of a binary, this extension complies to the RFC-7493, so this method assumes there is a String value with the key and it contains a Base64 encoded binary, which it decodes if found and returns. binaryValue()
