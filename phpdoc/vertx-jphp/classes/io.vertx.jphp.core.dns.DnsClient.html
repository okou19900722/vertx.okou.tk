lookup()

lookup( arg0,  arg1) : \io\vertx\jphp\core\dns\DnsClient

Try to lookup the A (ipv4) or AAAA (ipv6) record for the given name. The first found will be used.

$name [string] the name to resolve

$handler [callable] the io.vertx.core.Handler to notify with the io.vertx.core.AsyncResult. The handler will get notified with the resolved address if a record was found. If non was found it will get notifed with {@code null}. If an error accours it will get failed.

resolveMX()

resolveMX( arg0,  arg1) : \io\vertx\jphp\core\dns\DnsClient

Try to resolve the MX records for the given name.

$name [string] the name for which the MX records should be resolved

$handler [callable] the io.vertx.core.Handler to notify with the io.vertx.core.AsyncResult. reverseLookup( arg0,  arg1) : \io\vertx\jphp\core\dns\DnsClient

$ipaddress [string] the IP address to resolve the PTR for
