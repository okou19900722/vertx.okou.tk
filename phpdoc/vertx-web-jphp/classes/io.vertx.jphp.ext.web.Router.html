accept()

accept( arg0) :

This method is used to provide a request to the router. Usually you take request from the {@link io.vertx.core.http.HttpServer#requestHandler(Handler)} and pass it to this method. The router then routes it to matching routes.

This method is now deprecated you can use this object directly as a request handler, which means there is no need for a method reference anymore.

$request [HttpServerRequest] the request
