receivedFrameHandler()

receivedFrameHandler( arg0) : \io\vertx\jphp\ext\stomp\StompClientConnection

Configures a received handler that get notified when a STOMP frame is received by the client.

This handler can be used for logging, debugging or ad-hoc behavior. The frame can still be modified by the handler.

Unlike StompClient#receivedFrameHandler(Handler), the given handler won't receive the {@code CONNECTED} frame. If a received frame handler is set on the StompClient, it will be used by all clients connection, so calling this method is useless, except if you want to use a different handler. $handler [callable] the handler receivedFrameHandler($handler)
