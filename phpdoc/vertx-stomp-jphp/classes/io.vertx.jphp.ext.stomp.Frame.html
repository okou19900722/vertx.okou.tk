\io\vertx\jphp\ext\stomp

\io\vertx\jphp\ext\stompFrame

Represents a STOMP frame. STOMP frames are structured as follows. It starts by a {@code command}, followed by a set of headers. Then the frame may have a body and is finished by a {@code 0} byte. This class represents this structure and provide access to the different parts.

This class is NOT thread-safe.
