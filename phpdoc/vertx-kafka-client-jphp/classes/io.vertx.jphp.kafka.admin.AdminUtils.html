createTopic($topicName,$partitionCount,$replicationFactor,$completionHandler)

Creates a new Kafka topic on all Brokers managed by the given Zookeeper instance(s). In contrast to @see #createTopic(String,, one can pass in additional configuration parameters as a map (String -> String).

$topicName [string] Name of the to-be-created topic

$partitionCount [integer] Number of partitions

$replicationFactor [integer] Number of replicates. Must be lower or equal to the number of available Brokers

$topicConfig [array] map with additional topic configuration parameters
